# zoomies

<!-- badges: start -->
<!-- badges: end -->

The goal of zoomies is to provide access to the data set from the
article [Reliability and Validity of Seven Feline Behavior and
Personality Traits](https://www.mdpi.com/2076-2615/11/7/1991).

## Installation

You can install the development version of zoomies from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("EmilHvitfeldt/zoomies")
```
